# ふらのジャム園 — 歴史と3Dジャム瓶（パスワード保護）

閲覧用の共有パスワードで保護された静的サイトです。
各ページを開くとパスワード入力画面が表示され、正しいパスワードを入力した場合のみ内容が表示されます。

🔗 https://que-dotcom.github.io/furano-jam-garden/

収録ページ:
- トップ（`index.html`）
- ふらのジャム園の歴史（`site/furanojam-history.html`）
- 果物型ジャム瓶 3D（`3d/index.html` ／ プレビュー2種）

> 内容は StatiCrypt によりブラウザ内で AES-256 復号されます。このリポジトリに置かれている HTML は暗号化済みで、パスワードなしでは読めません。3Dモデルの原本データ（STL/STEP）・生成スクリプトはリポジトリに含めていません。
